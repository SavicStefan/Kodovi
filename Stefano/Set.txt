Set:

insert - Insert element
erase - Erase elements
swap - Swap content
clear - Clear content
size - Return container size

find - Get iterator to element 
count - Count elements with a specific value 
lower_bound - Return iterator to lower bound ( prvi >= element)
upper_bound - Return iterator to upper bound ( prvi > element)
equal_range - Get range of equal elements

next(c) - sledeci element u set-u posle c, (c je interator, i think)
prev(c) - prethodni element u set-u pre c, (c je interator, i think)
