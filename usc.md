# Key

 * 1DRQ = 1D Range Query
 	* BIT = Binary Indexed Tree
 * DP = Dynamic Programming
 * PIE = Principle of Inclusion and Exclusion
 * Graphs
 	* DSU = Disjoint Set Union
 		* MST = Min Spanning Tree
 	* SP = Shortest Path
 	* TopoSort = Topological Sort
